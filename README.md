FINEnce

Description

FINEnce is a simple Java-based personal money management application.
It helps users track income and expenses, analyze spending by category, and understand their financial habits.

The application is designed to be lightweight, easy to use, and offline, focusing on core functionality without unnecessary complexity.

Project Goals
♥ Track income and expenses
♥ Show how much money is earned and spent
♥ Identify which life categories require the most money
♥ Provide a clear overview of personal finances

Technologies Used
♥ Java
♥ IntelliJ IDEA (JetBrains)
♥ Git & GitHub
♥ SQLite
♥ JDBC

Main Classes and Entities
♥ Transaction – represents income or expense
♥ Category – represents a spending category
♥ TransactionManager – manages transactions and calculations
♥ CategoryAnalyzer – analyzes spending by category
♥ DatabaseService – handles database operations
♥ Main – application entry point

MVP Features
♥ Adding income and expenses
♥ Adding teg of category to each transaction
♥ Viewing a list of transactions
♥ Calculations (total income, expenses, current balance)
♥ Statistics (the most expensive category, money flow)
♥ Storing and loading data from the database
